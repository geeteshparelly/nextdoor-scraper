## Nextdoor Scraper Project

Created `nextdoor-scraper/` project with browser-based scraping (no Playwright).

**Purpose:** Extract leads from Nextdoor by finding:
- Posts where people need services (potential clients)
- Business page links in comments (potential competitors/partners)

**Files Created:**
- `scraped_posts.json` - Full post data
- `scraped_ids.json` - Dedup tracking
- `scraper_functions.js` - Reusable JS snippets for browser evaluate()
- `SCRAPER_PROCESS.md` - Full documentation
- `scrape_log.md` - Run history

**Test Run Results:**
- 10 posts successfully scraped
- 0 business page links (small sample)
- Process works smoothly

**Key Selectors:**
- Post body: `[data-testid="post-body"]`
- Reply button: `[data-testid="post-reply-button"]`
- Comment container: `.comment-container`
- Business links: `a[href*="/page/"]`
- Close button: `[aria-label="Close expanded post"]`
